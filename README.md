# ComputerVision
Personal Repository for the ComputerVision(26spring, Fudan University)

## Environment Setup
This project requires Python and the following dependencies:
- torch
- torchvision
- numpy
- matplotlib
- sklearn
- tqdm

## HW1
Homework 1 focuses on implementing and training a neural network for image classification using the FashionMNIST dataset. The tasks include:
- Preprocessing the dataset
- Designing the model architecture
- Training the model
- Evaluating the model's performance

### Training
To train the model, run the following command:
```bash
python hw1/train.py
```

### Testing
To test the model, run the following command:
```bash
python hw1/test.py
```
